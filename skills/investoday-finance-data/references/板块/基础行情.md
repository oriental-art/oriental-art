#### 行业的最新实时日行情

接口：`get_industry_realtime_quote`　
请求方式: `GET`　
tool_id: `get_industry_realtime_quote`

输入行业代码，获取该行业的最新实时指数、涨跌幅、成交量、总市值、成分股数量、涨停/上涨/下跌/平盘/总股数、涨跌幅排名、领涨股等关键行情数据，用于实时跟踪行业整体表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | ✅ | string | 行业代码 | `330000` |

**接口示例**

```bash
python skills/scripts/call_api.py industry-quote/realtime industryCode=330000
```

#### 概念的最新实时日行情

接口：`get_concept_realtime_quote`　
请求方式: `GET`　
tool_id: `get_concept_realtime_quote`

输入概念类型（如聚源、财联社）和概念代码，获取该概念板块的最新实时行情数据，包括板块名称、涨跌幅、一周涨跌幅、总市值、成分股数量、涨停/上涨/下跌/平盘股数、涨跌幅排名、领涨股信息等关键指标，便于追踪特定市场热点板块表现。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `conceptType` | ✅ | string | 概念类型(jy-聚源、cls-财联社) | `jy` |
| `conceptCode` | ✅ | string | 概念代码 | `14060061` |

**接口示例**

```bash
python skills/scripts/call_api.py concept-quote/realtime conceptType=jy conceptCode=14060061
```

#### 行业预测

接口：`list_industry_forecasts`　
请求方式: `GET`　
tool_id: `list_industry_forecasts`

通过行业代码获取行业预测数据，包括当前及未来几年的净利润增长率、业务收入增长率及其市场排名，以及历史净利润和业务收入数据，适用于行业趋势分析和投资决策支持。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryCode` | ✅ | string | 行业代码 | `640000` |

**接口示例**

```bash
python skills/scripts/call_api.py industry/forecasts industryCode=640000
```

#### 行业的最新实时日行情V2

接口：`list_ind_real_quote_v2`　
请求方式: **`POST`**　
tool_id: `list_ind_real_quote_v2`

通过指定行业代码（可批量，空数组表示全市场）、行业等级、行业分类（如申万SW）等条件，获取行业的最新实时行情数据，包括行业价格、涨跌幅、近一周涨跌幅、成交量、总市值、数据时间（格式为yyyy-MM-dd HH:mm:ss）、行业内股票涨跌家数、领涨股及ETF信息、行业排名等核心指标，并支持按指定字段（如涨跌幅）和排序方式（升序/降序）进行分页查询，适用于实时监测行业板块表现、分析市场热点和进行行业对比等任务。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `industryLevel` | — | integer | 行业等级(1、2、3)行业代码为空时必传 | `1` |
| `industryType` | — | string | 行业类型(默认: SW) | `SW` |
| `order` | — | string | 排序方式(asc、desc) | `desc` |
| `sortColumn` | ✅ | string | 排序字段(接口返回值) | `changeRatio` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
| `industryCodes` | — | array | 行业代码(全市场为:[]) | — |

**接口示例**

```bash
# 可选参数: industryLevel, industryType, order, pageSize, industryCodes
python skills/scripts/call_api.py industry-quote/realtime-v2 --method POST sortColumn=changeRatio
```

#### 概念的最新实时日行情V2

接口：`list_concept_real_quote`　
请求方式: **`POST`**　
tool_id: `list_concept_real_quote`

根据聚源或财联社的概念类型，查询指定概念代码（可批量）的最新实时行情数据，包括概念名称、涨跌幅、近一周涨跌幅、概念内股票涨跌家数、涨停家数、总市值、领涨股票及ETF信息、数据更新时间（yyyy-MM-dd HH:mm:ss），并支持按指定字段（如涨跌幅）进行排序和分页，用于实时监测概念板块的整体表现和热点追踪。

**输入参数**

| 参数名 | 必填 | 类型 | 说明 | 示例 |
|--------|:----:|------|------|------|
| `conceptType` | ✅ | integer | 概念类型 | `1` |
| `order` | — | string | 排序方式(asc、desc) | `desc` |
| `sortColumn` | ✅ | string | 排序字段(接口返回值) | `changeRatio` |
| `pageSize` | — | integer | 页长。 最小值:1; 最大值:500; | `10` |
| `conceptCodes` | — | array | 概念代码(全市场为:[]) | — |

**接口示例**

```bash
# 可选参数: order, pageSize, conceptCodes
python skills/scripts/call_api.py concept-quote/realtime-v2 --method POST conceptType=1 sortColumn=changeRatio
```
