# A股中小盘科技股筛选分析报告

## 筛选条件回顾
1. 市值200亿以内（中小盘科技股）
2. 股价处于低位或严重超跌（相比历史高点回调较深）
3. 必须是科技股（半导体、AI、软件、电子、通信等）
4. 有独门技术，是特定领域的行业龙头或细分领域隐形冠军
5. 未来有可能受益于科技相关热点（AI、国产替代、机器人、自动驾驶等）
6. 近几个月机构密集调研，有研报关注
7. 筹码开始逐步集中（股东户数减少）
