## 🔀 数据流图谱

### 消息流入流程

```
用户发送消息
     │
     ▼
┌─────────────────┐
│  Channel Layer  │  ← WhatsApp/Telegram/Discord 等
│    渠道层        │
└────────┬────────┘
         │ 解析消息
         ▼
┌─────────────────┐
│  Gateway Core   │  ← 接收原始消息
│   Gateway核心   │
└────────┬────────┘
         │ 路由决策
         ▼
┌─────────────────┐
│ Session Manager │  ← 获取/创建会话
│   会话管理器     │
└────────┬────────┘
         │ 加载上下文
         ▼
┌─────────────────┐
│  Memory System  │  ← 读取历史记忆
│   记忆系统       │
└────────┬────────┘
         │ 构建提示
         ▼
┌─────────────────┐
│  Agent Runtime  │  ← 调用AI模型
│   Agent运行时    │
└────────┬────────┘
         │ 生成回复
         ▼
┌─────────────────┐
│  Tool Executor  │  ← 执行工具（如需要）
│   工具执行器     │
└────────┬────────┘
         │ 返回结果
         ▼
┌─────────────────┐
│  Channel Layer  │  ← 发送回复
│    渠道层        │
└────────┬────────┘
         │
         ▼
      用户收到
```

## 🏗️ 组件关系矩阵

### Gateway 内部组件

| 组件 | 依赖 | 被依赖 | 通信方式 |
|------|------|--------|----------|
| **Router** | - | SessionMgr, Agent | 内部调用 |
| **SessionMgr** | Router | Agent, Memory | 内部调用 |
| **ChannelMgr** | - | Router | WebSocket |
| **Agent** | SessionMgr, Memory | Tool Registry | 内部调用 |
| **Memory** | - | SessionMgr, Agent | 文件/DB |
| **Auth** | - | 所有组件 | 中间件 |

### 外部集成关系

| 组件 | 上游 | 下游 | 协议 |
|------|------|------|------|
| **WhatsApp** | Baileys | Gateway | WebSocket |
| **Telegram** | grammY | Gateway | HTTP/WebSocket |
| **Discord** | discord.js | Gateway | WebSocket |
| **Model Provider** | Gateway | OpenAI/Anthropic | HTTP API |
| **Browser** | Gateway | Chrome/CDP | CDP Protocol |
| **Skill** | Gateway | Node/Python | 进程间 |
